DGGS native data cubes

The geospatial data is directly stored along dimensions of a DGGS index to optimize space and time needed to store, load, and process the data.

Interoperability

Generated data cubes are stored in Zarr using xcube and CF conventions. This allows direct data access from other packages and languages, such as xarray in Python and stars in R.

Features

Conversion of coordinates and data between geographical and DGGS spaces, data visualization, Neighbor retrieval
